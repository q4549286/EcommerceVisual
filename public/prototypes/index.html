商品图生成器 · 设计原型对比

三种不同的布局与交互思路，选一个继续实现。

A

左右双栏（精简）

左侧固定表单 + 右侧结果。结构稳定，适合每次都从头填表的场景。
B

顶部表单 + 纵向流

紧凑顶栏一行内完成输入，每次生成作为一段卡片纵向流。像聊天/反馈流，适合连续生成多次。
C

画廊 + 抽屉

主页直接是历史画廊，点右下「+」打开生成抽屉。历史即首页，最像成熟工具产品。

+

每个原型都包含以下功能演示：

  • 简化的表单（仅图片 + 名称为必填）
  • 图片类型多选
  • 生成时全局遮罩 + 动态进度条 + 分步骤提示
  • 历史记录弹窗，展示过往生成
  • 调用日志弹窗，展示 API 请求与响应
  • 点击结果图片放大查看（lightbox）
