# PRD：AI Product Listing Image Generator

## 1. 产品定位

AI Product Listing Image Generator 是一个面向跨境电商和台湾电商卖家的轻量商品图生成工具。用户上传商品图并填写少量商品信息后，系统生成可用于上架的白底首图、商品亮点介绍图和细节/规格说明图。

## 2. 目标用户

- 跨境电商卖家
- 台湾电商卖家
- Shopify / DTC 独立站运营
- Amazon / eBay / Etsy / Shopee Taiwan 商品运营
- 小团队设计和运营人员

## 3. 目标市场

MVP 重点支持：

- Amazon / Marketplace
- Shopify / DTC
- Taiwan E-commerce

MVP 重点语言：

- English
- 繁體中文

## 4. MVP 范围

### 4.1 默认生成 3 类图

1. 白底首图：商品居中、纯白背景、无文字、适合平台首图。
2. 商品亮点介绍图：展示 3-5 个本地化卖点，适合副图。
3. 细节/规格说明图：展示材质、尺寸、包装、功能细节等信息。

### 4.2 可选生成 1 类图

4. 使用场景图：将商品置于真实生活或使用场景中，适合 Shopify、Etsy、台湾电商副图。

## 5. 非 MVP 范围

## 6. 用户流程

1. 上传商品图。
2. 选择目标平台、市场和输出语言。
3. 填写商品名称、类目、描述和可选卖点。
4. 选择图片类型。
5. 选择 Prompt 模式或 Generate 模式。
6. 点击生成。
7. 查看图片方案、Prompt、本地化文案和生成结果。
8. 下载图片或复制 Prompt。

## 7. 输入字段

### 必填

- 商品图
- 商品名称
- 商品类目
- 商品描述
- 目标平台
- 输出语言

### 选填

- 品牌名
- 材质
- 尺寸
- 颜色
- 目标人群
- 已有卖点
- 参考图
- 禁止出现内容

## 8. 输出内容

每个图片结果包含：

- 图片类型
- 用途说明
- 本地化图内文案
- 设计说明
- 生图 Prompt
- Negative Prompt
- 图片结果，如果启用 Generate 模式

## 9. 核心规则

### 首图规则

- 不加文案
- 不加 Logo
- 不加促销贴纸
- 不加水印
- 不加边框
- 使用白底或纯白背景
- 商品真实、完整、居中展示

### 亮点图规则

- 展示 3-5 个卖点
- 英文短句化
- 繁中自然表达
- 不生成无依据数据
- 不使用绝对化承诺

### 细节图规则

- 优先展示材质、尺寸、包装内容、局部细节、使用方式
- 信息清晰，不拥挤
- 不虚构认证、参数和功效

## 10. 生图接口

使用 OpenAI 兼容 Image API。

- Base URL：`IMAGE_API_BASE`（OpenAI 兼容的 `/v1` 接口）
- Model：`IMAGE_MODEL`
- API Key：`IMAGE_API_KEY`，可在 `.env` 或管理后台“系统设置”中配置

接口优先级：

- 有商品图时使用 `/v1/images/edits`
- 无商品图时使用 `/v1/images/generations`

MVP 中用户必须上传商品图，因此主要使用编辑接口。

## 11. 成功标准

- PC 和移动端均可流畅使用。
- 用户可在 1 个页面内完成生成。
- 默认能生成 3 类图片方案。
- 没有 API Key 时可以正常使用 Prompt 模式。
- 配置 API Key 后可以使用 Generate 模式。
- 生成失败时有清晰错误提示。
