# UI 设计规范

## 1. 设计原则

界面采用极简风格，去除不必要的装饰、渐变与多色块。所有色彩仅在黑、白、灰之间。PC 与移动端共用同一套规则。

## 2. 色彩

- 文字主色：#0f172a
- 文字次色：#64748b
- 边框：#e2e8f0
- 背景：#ffffff
- 选中态：#0f172a 反白

不使用渐变、彩色徽标、多色按钮。

## 3. 字体

- 字体栈：系统默认字体
- 标题：18-22px，font-semibold
- 正文：13-14px，font-normal
- 辅助：11-12px，text-slate-500

界面文案统一使用简体中文，不混用中英文。

## 4. 间距

- 页面外边距：16-24px
- 模块间距：24px
- 字段间距：12-16px

## 5. 组件

- 按钮：黑底白字或白底黑字，圆角 4px
- 输入框：白底，1px 灰边，聚焦时变黑边
- 卡片：白底，1px 灰边，无阴影
- 选项按钮：未选灰边，选中黑底白字

不使用大圆角、不使用阴影、不使用动画过渡之外的特效。

## 6. 响应式

- 桌面：左侧表单 380-400px 固定宽度，右侧结果区自适应
- 移动：单栏，从上到下依次为表单、生成按钮、结果

## 7. 图标

不引入图标库。复选与勾选直接用文字符号 ✓ 表达。
